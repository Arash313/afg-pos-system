# Business Rules

1. Product barcode must be unique.

2. Product internal code must be unique.

3. Stock cannot become negative.

4. Every sale must create stock movement.

5. Every purchase must create stock movement.

6. Deleted products should be archived, not permanently removed.

7. Invoice numbers are generated automatically.

8. Backup files must include database and settings.

9. Every user action must be logged.

10. Low stock alert should be displayed on dashboard.
